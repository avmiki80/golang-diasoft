1. go install github.com/oapi-codegen/oapi-codegen/v2/cmd/oapi-codegen@latest

2. Команда установилась, но не найдена в PATH. Нужно добавить GOPATH/bin в PATH или использовать полный путь. Проверю где установлен Go:
go env GOPATH

3. Проверю, установился ли файл:
Test-Path C:\go\go1.23.12\bin\oapi-codegen.exe

4. Файл не установился в GOPATH. Проверю правильный GOBIN:
go env GOBIN

5. GOBIN не установлен. Проверю где Go устанавливает бинарники по умолчанию:
$env:USERPROFILE + '\go\bin\oapi-codegen.exe' | Test-Path
Get-ChildItem C:\go\go1.23.12\bin\ | Select-Object Name

6. Проверю в подпапке bin:
Get-ChildItem C:\go\go1.23.12\bin\bin\ -ErrorAction SilentlyContinue | Select-Object Name

7. Заапуск с правильным путем:
C:\go\go1.23.12\bin\bin\oapi-codegen.exe -help
